Comment je trie l’information

Je commence par les sources officielles pour eviter le bruit, puis je regarde si le sujet change vraiment quelque chose pour l’interface, le flux de travail, l’experience utilisateur ou l’architecture d’un produit. Ensuite je garde une note courte, datee et orientee usage.
